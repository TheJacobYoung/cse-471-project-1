CSE 471 - Project 1 - Effects (Evan Daykin)

Sample Xml Configuration

Effects are added on as attributes tto an instrument:

    <instrument instrument="Organ" chorus="0.6" flange="0.3" compression="0.1" noisegate="0.2">
  </instrument>

Grading Elements

  • 10 - Component passes audio
  • 20 - 1 Effect
  • 30 - 3 Effects
  • 40 - Controllable effects send
  • 50 - 4 Effects

NOTE: Effects were having problems with an out of memory error, so no WAV file was able to be produced.
